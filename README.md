# 🌳 大樹教練 · AI 特助心法（一鍵注入版）

> 26 年企業培訓 + 1.5 萬位老闆教練實戰沉澱出的 **AI 協作方法論**。
> 一頁網站 + 一份系統提示詞，讓任何 AI 在 30 秒內進入「老闆動嘴，AI 全自動執行」狀態。

## 🚀 線上版

**永久 URL**：https://coach1973.github.io/ai-go/

點「一鍵複製注入提示詞」按鈕，貼到任何 AI 對話框即可。

## 📐 設計理念

**問題**：每次換 AI 平台 / 開新對話 → 都要重新教 AI 怎麼當特助 → 摩擦巨大。

**解法**：把 26 年方法論濃縮成一份「一字不差」的注入版，跨 AI 跨裝置共用。

**特性**：
1. 一份 markdown 打死，跨 AI 兼容
2. 內建「驗證題」確保 AI 真的讀進去
3. 內建「開工宣言」儀式感 + 跨對話一致性
4. CC BY-ND 確保心法不被改寫
5. 完全公開，可放 LinkedIn / IG / 名片 QR Code
